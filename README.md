# Thailand Tourism Data Analysis (2022–2024)

## Objective
โปรเจคนี้มีวัตถุประสงค์เพื่อวิเคราะห์ข้อมูลการท่องเที่ยวภายในประเทศของประเทศไทย
ในช่วงปี 2022–2024 เพื่อศึกษาภาพรวมแนวโน้มการท่องเที่ยว ลักษณะเชิงฤดูกาล
และจังหวัดที่มีบทบาทสำคัญต่ออุตสาหกรรมการท่องเที่ยว

## Dataset
- แหล่งข้อมูล: Kaggle – Thailand Domestic Tourism
- ข้อมูลครอบคลุม: ปี 2022–2024
- ระดับข้อมูล: รายจังหวัด รายเดือน
- ตัวแปรหลัก:
  - year, month
  - province, region
  - tourists
  - revenue_million_thb

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Data Cleaning
- ตรวจสอบและจัดการ missing values
- ลบข้อมูลที่ซ้ำซ้อน (duplicate records)
- ตรวจสอบชนิดข้อมูลให้ถูกต้อง
- ตรวจสอบความสมเหตุสมผลของข้อมูลรายเดือนและรายปี
- จัดรูปแบบชื่อจังหวัดและภูมิภาคให้สอดคล้องกัน

## Analysis Process
1. วิเคราะห์ภาพรวมการท่องเที่ยวรายปี (Trend)
2. วิเคราะห์ High Season และ Low Season รายเดือน
3. วิเคราะห์จังหวัดที่มีนักท่องเที่ยวและรายได้สูงสุด
4. สรุป Insight เชิงธุรกิจจากข้อมูล

## Key Insights
- การท่องเที่ยวไทยมีแนวโน้มฟื้นตัวอย่างต่อเนื่องในช่วงปี 2022–2024
- การท่องเที่ยวมีลักษณะเชิงฤดูกาลอย่างชัดเจน โดยช่วงปลายปีเป็น High Season
- จังหวัดชลบุรี ขอนแก่น กรุงเทพฯ เชียงใหม่ และภูเก็ต
  เป็นจังหวัดที่มีจำนวนนักท่องเที่ยวสูงสุด
- จังหวัดที่มีนักท่องเที่ยวจำนวนมากไม่ได้หมายความว่าจะมีรายได้สูงสุดเสมอไป

## Conclusion
โปรเจคนี้แสดงให้เห็นถึงการใช้ข้อมูลจริงในการวิเคราะห์เชิงธุรกิจ
ตั้งแต่การทำความสะอาดข้อมูล การวิเคราะห์แนวโน้ม
ไปจนถึงการสรุป Insight ที่สามารถนำไปใช้ประกอบการตัดสินใจได้
